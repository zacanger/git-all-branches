# ts-cli-boilerplate

Boilerplate for simple TypeScript command-line tools.

--------

## Usage

* Click the "Use this template button" to create a new repo.
* Clone your repo
* `npm ci`
* Write some code!
* `npm run build`

Check out the scripts and hooks in package.json for details.
